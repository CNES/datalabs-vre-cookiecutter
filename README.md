# Template Cookiecutter for VRE  
  
## Usage
Execute cookiecutter (you need an environment with cookiecutter)  
cookiecutter https://github.com/CNES/datalabs-vre-cookiecutter.git

Cookie-cutter will ask you several questions :  

  [1/5] Name of your VRE (Cookiecutter VRE Communautaire): MyVRE  
  [2/5] Slug of your VRE (no space) (myvre):   
  [3/5] Project's author (CNES): Me  
  [4/5] Do you want to use the pangeo notebook layer (needed for pytorch layer)? [y/n] (n): y  
  [5/5] Do you want to use the pytorch notebook layer (need pangeo layer)? [y/n] (n): n  

You will be proposed to use all available modules.  

Your VRE is available in myvre folder !
